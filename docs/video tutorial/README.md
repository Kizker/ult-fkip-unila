# Paket Naskah Video Tutorial Pemohon

Folder ini berisi paket lengkap materi video tutorial untuk pemohon layanan seperti mahasiswa.

## Struktur Folder
- `01-pengenalan-sistem-layanan.md`
- `02-daftar-login-dan-lengkapi-profil.md`
- `03-cara-mengajukan-layanan.md`
- `04-status-revisi-dan-unduh-hasil.md`
- `narasi formal/`
- `narasi natural/`
- `storyboard/`

## Keterangan
- File di folder utama berisi naskah lengkap dasar per video.
- Folder `narasi formal` berisi versi voice over yang lebih resmi.
- Folder `narasi natural` berisi versi voice over yang lebih ringan dan enak dibacakan.
- Folder `storyboard` berisi urutan scene per video untuk kebutuhan rekam dan editing.
