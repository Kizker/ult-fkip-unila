# Storyboard Video 2

## Judul
Panduan Mahasiswa: Cara Daftar, Login, dan Melengkapi Profil

## Scene
1. Opening title
Visual: Judul video tampil.
Narasi: Pada video ini akan dijelaskan langkah-langkah pendaftaran akun, proses login, serta cara melengkapi profil pengguna.

2. Halaman registrasi
Visual: Form pendaftaran akun dibuka.
Narasi: Langkah pertama adalah membuka halaman pendaftaran akun.

3. Pengisian data registrasi
Visual: Kursor mengisi nama, email, NIM, dan kolom lain.
Narasi: Pada halaman ini, pengguna diminta mengisi data identitas sesuai informasi yang sebenarnya.

4. Sorot ketelitian pengisian
Visual: Zoom pada kolom form penting.
Narasi: Isilah nama lengkap, alamat email, nomor induk mahasiswa, dan data lain yang diminta oleh sistem secara benar dan teliti.

5. Pembuatan password
Visual: Kursor mengisi password.
Narasi: Setelah seluruh data terisi, buatlah password untuk akun yang akan digunakan.

6. Kirim formulir pendaftaran
Visual: Klik tombol daftar atau registrasi.
Narasi: Jika seluruh data telah benar, klik tombol daftar atau registrasi untuk mengirimkan pendaftaran akun.

7. Halaman login
Visual: Pindah ke form login.
Narasi: Setelah proses registrasi berhasil, pengguna dapat melanjutkan ke halaman login.

8. Proses login
Visual: Isi email dan password, lalu klik masuk.
Narasi: Masukkan email atau akun yang telah didaftarkan, kemudian masukkan password yang sesuai.

9. Dashboard pemohon
Visual: Dashboard muncul setelah login berhasil.
Narasi: Apabila data login benar, pengguna akan diarahkan ke dashboard pemohon.

10. Menu profil
Visual: Kursor membuka menu profil atau pengaturan akun.
Narasi: Sebelum mengajukan layanan, pengguna disarankan untuk membuka menu profil dan memeriksa kelengkapan data diri terlebih dahulu.

11. Pemeriksaan data profil
Visual: Sorot data pribadi dan data akademik.
Narasi: Pastikan data pribadi dan data akademik telah terisi dengan benar.

12. Unggah dokumen pendukung
Visual: Sorot area unggah file jika tersedia.
Narasi: Dalam kondisi tertentu, sistem juga dapat meminta pengguna untuk mengunggah dokumen pendukung.

13. Simpan perubahan
Visual: Klik tombol simpan profil.
Narasi: Setelah seluruh data profil diperiksa dan dilengkapi, simpan perubahan yang telah dilakukan.

14. Penutup
Visual: Profil dalam kondisi lengkap.
Narasi: Setelah proses pendaftaran, login, dan pelengkapan profil selesai, pengguna telah siap untuk mengajukan layanan melalui sistem.
