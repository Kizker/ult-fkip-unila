# Template Analisis Aiken's V

Dokumen ini disusun untuk menganalisis hasil validasi ahli materi terhadap:

- `instrumen-uji-validitas-ahli-materi-v3.docx`

Jumlah butir: `22`

Skala penilaian:

- `1 = tidak sesuai`
- `2 = kurang sesuai`
- `3 = cukup sesuai`
- `4 = sesuai`
- `5 = sangat sesuai`

## Rumus Aiken's V

Rumus yang digunakan:

`V = sum s / [n (c - 1)]`

Keterangan:

- `V` = koefisien validitas Aiken
- `s` = `r - lo`
- `r` = skor yang diberikan validator
- `lo` = skor terendah pada skala penilaian
- `n` = jumlah validator
- `c` = jumlah kategori skor

Pada instrumen ini:

- `lo = 1`
- `c = 5`

Sehingga:

- `s = r - 1`

## Interpretasi Praktis

Interpretasi dapat menggunakan kategori berikut:

- `V < 0,40` = validitas rendah
- `0,40 <= V < 0,80` = validitas sedang
- `V >= 0,80` = validitas tinggi

Untuk kebutuhan keputusan revisi skripsi, aturan kerja yang aman:

- `V >= 0,80` = item valid
- `0,60 - 0,79` = item valid dengan revisi kecil
- `< 0,60` = item perlu revisi besar atau dipertimbangkan untuk diganti

## Cara Menghitung

Jika jumlah validator adalah `3`, maka:

- `n = 3`
- `c - 1 = 4`
- penyebut = `3 x 4 = 12`

Langkah hitung:

1. Jumlahkan skor mentah dari seluruh validator untuk satu butir.
2. Hitung `sum s` dengan menjumlahkan `(r - 1)` dari seluruh validator.
3. Hitung `V = sum s / 12`.

## Contoh Perhitungan

Contoh pada satu butir:

- Validator 1 = `5`
- Validator 2 = `4`
- Validator 3 = `5`

Maka:

- `s1 = 5 - 1 = 4`
- `s2 = 4 - 1 = 3`
- `s3 = 5 - 1 = 4`
- `sum s = 11`
- `V = 11 / 12 = 0,92`

Interpretasi:

- butir memiliki validitas tinggi
- butir dapat dinyatakan valid
