## Aturan Analisis

Rumus:

`Persentase kepraktisan = (skor diperoleh / skor ideal) x 100%`

Keterangan:

- `skor diperoleh` = jumlah seluruh skor jawaban responden
- `skor ideal` = jumlah butir x skor maksimum x jumlah responden

Kategori:

- `81% - 100%` = sangat praktis
- `61% - 80%` = praktis
- `41% - 60%` = cukup praktis
- `21% - 40%` = kurang praktis
- `0% - 20%` = tidak praktis

## Catatan Penggunaan

- Instrumen ini digunakan sebagai instrumen umum lintas peran.
- Jika peneliti tetap ingin membahas buku panduan atau video tutorial, masukan tersebut cukup dicatat pada bagian komentar dan saran umum, bukan dijadikan butir inti penilaian.
