Instrumen ini digunakan untuk menilai tingkat kepraktisan Website Unit Layanan Terpadu (ULT) FKIP Universitas Lampung dari berbagai peran pengguna.

### Petunjuk Pengisian

1. Responden diminta mencoba Website ULT sesuai dengan peran masing-masing sebelum mengisi angket.
2. Bacalah setiap pernyataan dengan cermat.
3. Berikan tanda centang pada salah satu pilihan jawaban yang paling sesuai dengan pengalaman penggunaan.
4. Semua butir pernyataan wajib diisi.
5. Jika terdapat masukan tambahan, tuliskan pada bagian komentar dan saran umum.

### Skala Penilaian

- `5 = Sangat Setuju`
- `4 = Setuju`
- `3 = Netral`
- `2 = Tidak Setuju`
- `1 = Sangat Tidak Setuju`

## Aturan Analisis

Instrumen ini dianalisis menggunakan rumus persentase kepraktisan sebagai berikut:

`Persentase kepraktisan = (skor diperoleh / skor ideal) x 100%`

Keterangan:

- `skor diperoleh` adalah jumlah seluruh skor jawaban responden
- `skor ideal` adalah jumlah butir x skor maksimum x jumlah responden

Kategori interpretasi:

- `81% - 100%` = sangat praktis
- `61% - 80%` = praktis
- `41% - 60%` = cukup praktis
- `21% - 40%` = kurang praktis
- `0% - 20%` = tidak praktis
